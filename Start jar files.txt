1. Разбить имеющийся файл на несколько файлов (5-10 файлов) и залить в одну папку.
Код создает подпапку в папке с файлом, который подлежит разбивке. Подпапке присваивается <постоянная_часть_имени_полученных_файлов>

java -jar sep_file <имя_лог_файла> <постоянная_часть_имени_полученных_файлов> 
Например, 
java -jar sep_file.jar "F:\ИПР\TE\JAVA\results\mail.log.18.11.2014\main.log.2014-11-17" sep-files

2. Написать парсер логов, который выполняет следующие действия над логами в п.1:
   а) С помощью регулярного выражения задать фразу поиска и все найденные строки записать в новый файл

java -jar find_words.jar <фраза_поиска> <путь_до_лог_файла(ов)> <имя_нового_файла> 
Например, 
java -jar find_words.jar "WARN :" "F:\ИПР\TE\JAVA\results\mail.log.18.11.2014\sep-files" word_search.txt

   б) Вставить разделитель ";"  между значениями в строке во всем файле, например, 17.11.2014;11:43:51.347;TRACE:;9.6.25.83:54650;Count is 1157, и записать в новый файл с расширением csv 

java -jar search_result_to_csv.jar <тип_разделителя> <путь_до_лог_файла(ов)> <имя_нового_файла> 
Например, 
java -jar search_result_to_csv.jar ", " "F:\ИПР\TE\JAVA\results\mail.log.18.11.2014\sep-files\word_search" result
